# Akte 001 — Tic Tac UAP

## Thema
USS Nimitz / Tic Tac UAP Encounter

## Kernpunkte
- 2004, USS Nimitz Carrier Strike Group
- Cmdr. David Fravor and other Navy personnel reported unusual object behavior
- Object described as white, smooth, tic-tac shaped
- No visible wings, exhaust, or rotors
- Radar operators reportedly tracked unusual movements
- Event became one of the most cited modern UAP incidents
