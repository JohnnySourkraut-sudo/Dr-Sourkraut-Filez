## 🎨 Design System: V9 Dossier Format

### Visual Elements
- **Background:** Dark grid pattern (#1a1a1a) simulating document scanning
- **Border Style:** Dashed (#4a4a4a) with SCIF-aesthetic inset shadows
- **Typography:** Courier New monospace (typewriter aesthetic)
- **Color Accents:** Case-specific (yellow/gold, orange, blue, brown)
- **Classification Banners:** Distinct colors per classification level
- **Typewriter Accents:** Italicized quoted text with accent-color backgrounds
- **Section Headers:** ▶ arrow prefix, uppercase, letter-spacing
- **Document Folds:** Simulated paper fold corner effect
- **Assessment Boxes:** Green-border boxes for official findings

### Typography Scale
- Case Title: 32px, uppercase, text-shadow
- Section Header: 14px, uppercase, letter-spacing 2px
- Content Text: 12px, line-height 1.8
- Metadata: 11px, uppercase, letter-spacing 1px
- Footer Stamp: 10px, uppercase
